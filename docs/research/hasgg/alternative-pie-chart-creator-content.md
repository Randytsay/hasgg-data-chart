### Description Text
另類餅圖製作工具是一個用戶友好的免費在線工具，允許用戶輕鬆創建和自定義另類餅圖。無論您是分析數據、準備報告，還是在演示中展示信息，我們的工具都能幫助您快速生成具有吸引力的另類餅圖。享受簡單的操作流程、直觀的界面和豐富的自定義選項，為您的數據可視化提供完美解決方案。

### Feature List
- 基本信息輸入：支持可選輸入另類餅圖標題、副標題、統計維度、單位等。
- 數據輸入: 支持手動輸入數據，方便快捷。
- 圖表主題: 可選另類餅圖的顏色、圖例，以滿足不同需求。
- 附加信息及水印：可為另類餅圖添加附加信息在左下角，添加文字水印在右下角。
- 實時預覽: 用戶在調整設置時可以實時查看另類餅圖的變化，提高設計效率。
- 導出和分享: 支持將另類餅圖導出為PNG格式，便於分享和插入到報告中。
- 清除所有：可以清除當前所有輸入及製作，以便進行新的另類餅圖製作。
- 全程本地處理: 純本地處理，完全安全，無需擔心隱私問題。
