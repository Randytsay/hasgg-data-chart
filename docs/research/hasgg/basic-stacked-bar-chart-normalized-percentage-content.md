### Description Text
基礎堆疊條形圖歸一化百分比製作工具是一款免費在線工具，用戶可以輕鬆創建、定製和下載歸一化為百分比的基礎堆疊條形圖。支持多種主題和顏色選擇，用戶可以通過填寫數據或選擇Excel表格實時生成專業的圖表，適用於各種行業的數據可視化需求。

### Feature List
- 圖表生成：根據用戶輸入的數據實時將基礎堆疊條形圖歸一化為百分比。
- 多主題支持：提供多種主題選擇以美化圖表外觀。
- 數據輸入：通過互動式表格錄入數據，簡單、直觀。
- 數據導入：通過選擇Excel文件導入數據。
- 自定義選項：允許用戶輸入標題、副標題、統計維度和計量單位，添加附加信息和水印。
- 圖表下載：支持將生成的圖表下載為PNG格式圖片。
- 圖例位置調整：可選擇圖例顯示的位置，包括左側、右側、頂部、底部和隱藏。
- 數據添加和刪除：用戶可以靈活地添加或刪除數據項。
- 自動生成：在頁面載入時自動生成一個示例歸一化為百分比的基礎堆疊條形圖，方便用戶使用。
- 實時預覽：實時查看數據變化對圖表的影響。
- 清除功能：清除所有輸入的數據和選項，重新開始。
- 全程本地處理: 純本地處理，完全安全，無需擔心隱私問題。
