## Description Section

氣泡圖製作工具是一個強大的在線平台，幫助用戶快速創建和自定義氣泡圖，從而有效地可視化多維數據。憑藉其直觀的用戶界面和豐富的自定義選項，用戶可以上傳數據，快速生成專業的氣泡圖，適合用於商業分析、學術研究和教育展示。無論您是數據分析師、學者，還是需要製作報告的學生，這款工具都能助您輕鬆展示複雜數據關係。

### Features List
- 圖表生成：根據用戶輸入的數據實時生成氣泡圖。
- 數據輸入：通過互動式表格錄入數據，簡單、直觀。
- 數據導入：通過選擇Excel文件導入數據。
- 自定義選項：允許用戶輸入標題、副標題，添加附加信息和水印。
- 圖表下載：支持將生成的圖表下載為PNG格式圖片。
- 圖例位置調整：可選擇圖例顯示的位置，包括左側、右側、頂部、底部和隱藏。
- 數據添加和刪除：用戶可以靈活地添加或刪除數據項。
- 自動生成：在頁面載入時自動生成一個示例氣泡圖，方便用戶使用。
- 實時預覽：實時查看數據變化對圖表的影響。
- 清除功能：清除所有輸入的數據和選項，重新開始。
- 全程本地處理: 純本地處理，完全安全，無需擔心隱私問題。
