### Feature Description
桑基圖製作工具是一款免費在線工具，用戶可以輕鬆創建、定製和下載桑基圖。支持多種主題和顏色選擇，用戶可以透過填寫數據或選擇Excel表格即時生成專業的圖表，適用於各行業的數據視覺化需求。

功能列表：
- 圖表生成：根據用戶輸入的數據實時生成桑基圖
- 多主題支援：提供多種主題選擇以美化圖表外觀
- 數據輸入：通過互動式表格錄入數據，簡單、直觀
- 數據匯入：通過選擇Excel文件導入數據
- 自訂選項：允許用戶輸入標題、副標題，添加附加信息和水印
- 圖表下載：支持將生成的圖表下載為PNG格式圖片
- 桑基圖類型：支持基礎和垂直方向的桑基圖
- 數據添加和刪除：用戶可以靈活地添加或刪除數據項
- 自動生成：在頁面載入時自動生成一個示例桑基圖
- 即時預覽：實時查看數據變化對圖表的影響
- 清除功能：清除所有輸入的數據和選項
- 全程本地處理: 純本地處理，完全安全
